Welcome to Maze Wars!

This is a web-based Maze Wars game inspired by the version written for Macintosh, circa 1987, author unknown. That, in turn, was inspired by the first Maze War game, which was created in 1973 by Steve Colley, Howard Palmer and Greg Thompson. This implemetation is released under the GNU General Public License. The source code can be found at the project's home on SourceForge.

This version is as true to the quick, simple gameplay of the Macintosh Maze Wars as fuzzy memories, and some surviving graphics files, could make it. It is easy to imagine many “improvements”, but Maze Wars seems to work best when the game play is kept simple and fast.
